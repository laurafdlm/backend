Servicios ofrecidos por el controlador de usuarios

Este sistema expone un controlador denominado "users" que ofrece servicios falsos de registro, identificación y eliminación de cuentas de usuario.

Toda la información se mantiene en memoria. Las cuentas de usuario se eliminan automáticamente entre 10 y 20 minutos después de haber sido creadas.

Registro de Usuario (Método 1)

POST /users/registrar1

Descripción: Permite registrar un usuario enviando un objeto JSON en el cuerpo de la petición con el email y las contraseñas.

Parámetros:

  • email: Correo electrónico del usuario.
  • pwd1: Contraseña del usuario.
  • pwd2: Confirmación de la contraseña.

Registro de Usuario (Método 2)

GET /users/registrar2?email=pepe@pepe.com&pwd1=Pepe1234&pwd2=Pepe1234

Descripción: Permite registrar un usuario utilizando parámetros en la URL.

Parámetros:

  • email: Correo electrónico del usuario.
  • pwd1: Contraseña del usuario.
  • pwd2: Confirmación de la contraseña.

Iniciar Sesión (Método 1)

PUT /users/login1

Descripción: Permite iniciar sesión enviando un objeto JSON en el cuerpo de la petición con los campos email y pwd.
Retorna un token de autenticación en el cuerpo de la respuesta. Este token debe usarse para eliminar la cuenta.

Iniciar Sesión (Método 2)

GET /users/login2?email=pepe@pepe.com&pwd=Pepe1234

Descripción: Permite iniciar sesión utilizando parámetros en la URL. Devuelve el usuario y un token en el encabezado de la respuesta. Este token debe usarse para eliminar la cuenta.

Iniciar Sesión (Método 3)

GET /users/login3/pepe@pepe.com?pwd=Pepe1234

Descripción: Permite iniciar sesión pasando el email como parte de la URL. La contraseña va en un parámetro pwd. Devuelve el usuario y un token en el encabezado de la respuesta. Este token debe usarse para eliminar la cuenta.

Obtener todos los Usuarios

GET /users/getAllUsers

Descripción: Devuelve una lista de todos los usuarios registrados.

Eliminar Usuario

DELETE /users/delete?email=pepe@pepe.com&pwd=Pepe1234

Descripción: Elimina un usuario si se proporciona el email y la contraseña correctos, y si el token enviado en la cabecera "token" es válido.

Eliminar Todos los Usuarios

DELETE /users/clearAll

Descripción: Elimina todos los usuarios del sistema. Debe llevar una cabecera "prime" con un nº primo de 3 cifras.
